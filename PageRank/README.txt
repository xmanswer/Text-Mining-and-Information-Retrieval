This is a script consists of various PageRank algorithm implementations. 
Locate this program in the same folder of files such as doc_topics.txt
Also locate the indri-lists folder in the same folder of this program
This program will parse all txt files and generate neccessary data structures
It will perform options with No-search-score (NS), weighted-search-score (WS) and 
customized-search-score (CS) on General PageRank (GPR), Query-Based-Topic-Sensitive-PageRank
(GTSPR) and Personalized-Topic-Sensitive-PageRank (PTSPR), generating nine
different output files. It will also generate three files for submission.
